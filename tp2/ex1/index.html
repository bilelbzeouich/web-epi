introduction au css

introduit au milieu des années 1990. CSS devient courament utilisé dans la co,ception de sites web et bien pris en charge par les navigatuers web dans les année 2000

les CSS est un langage de style qui définit la présantation des document HTML .Par exemple , CSS couvre les polices, les couleurs, le marges,les lignes, la hauteur,la largeur les images d'arriére-plan, les positionnements évolués et bien d'autres choses.

les but d'avoir les css dans vos sites c'est de compenser les manques du langage HTML en ce qui concerne la mise en page et la présentation
